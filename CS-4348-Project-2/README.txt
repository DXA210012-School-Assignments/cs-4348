=======================
HOTEL SIMULATION PROJECT
=======================

COMPILATION:
------------
To compile the program, use the following command:

javac HotelSimulator.java 


RUNNING THE PROGRAM:
--------------------
To run the program after compiling, use the command:

java HotelSimulator


ADDITIONAL DETAILS:
-------------------

1. The program uses threads and might produce slightly varied output on different runs due to the nature of threading.
2. If you encounter any unexpected behaviors, such as printing "Guest # joined" after printing "Simulation ends,"
    please try running the program again.
3. All the necessary files should be included in the provided zip file. If something seems missing, please reach out.
